# irresponsibility
Визуализация гипотезы о разделении женщин по степени безответственности <br>
test_data2.py - формирует набор тестовых данных <br>
plot2.py - визуализирует данные <br>


Ключевые особенности AI-подхода:
1. Реалистичные корреляции
Отрицательная корреляция между временем работы и временем дома

Зависимости между часами работы и уровнем безответственности

2. Профессиональные кластеры
Разные паттерны для офисных работников, медиков, учителей и т.д.

Кластеризация K-means для группировки схожих профилей

3. Реалистичное распределение имен
Частые имена (Анна, Мария) встречаются чаще

Редкие имена встречаются реже

4. Статистические модели
Нормальные распределения для основных показателей

Реалистичный шум и выбросы (5% данных)

5. Проверка согласованности
Сумма часов работы и дома не превышает 24 часа

Корректировка противоречивых данных

6. Анализ качества
Автоматический анализ статистик и корреляций

Валидация реалистичности данных

Программа создает данные, которые максимально приближены к реальным статистическим распределениям и зависимостям, встречающимся в настоящих демографических и поведенческих исследованиях!
